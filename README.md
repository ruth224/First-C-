## Overview of basics 

`Console.Writeline` -  Prints the text (Main code command) <br>
`Console.Write` - Prints text expecting user input <br>
`Console.Readline` - Reads users input returns as a string  <br>
`Console.Readkey` - Closes program <br>
`Console.clear` - Clears written code above and goes to the next line <br>
